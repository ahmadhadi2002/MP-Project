What is Rainbow Table Cracking?

Rainbow Table Cracking involves the usage of a rainbow table to crack a password. In modern day applications and databases, passwords are not stored in plaintext, but instead are hashed. How the cracking works, is very similar to dictionary attacks, where it is a simple comparison to stored hashes within a database. Once is match is found, the password is cracked.
